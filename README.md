# RGG — Дашборд игроков

## Взаимодействия

Таблица всех действий между игроками. Показывает дату, отправителя, получателей, тип действия и примечание.

**Фильтры:**
- Дата с / Дата по
- От кого / Кому (можно поменять местами)
- Тип действия
- Поиск по тексту
- Только активные игроки
- Закрепление фильтров при прокрутке

## Инвентари

Содержимое инвентарей активных игроков.

**Возможности:**
- Поиск по названию или описанию предмета
- Выбор конкретного предмета — покажет у кого он есть и в каком количестве
- Фильтрация по типу: эффекты, предметы, спецроллы
- Два режима: «Сумма» (общее количество) и «По игрокам»
- Панель «Общая информация» — монетки, слёзы, эффекты, предметы, спецроллы каждого игрока
- Клик по имени игрока открывает его инвентарь
